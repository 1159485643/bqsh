# 明星同款舆情风险复审 V3 - 应用身份直连表格版

## 本版核心变化

- 不再使用飞书 OAuth 登录。
- 不再检查 user_access_token。
- App ID / App Secret 已合并到“数据源设置”。
- 用户填写 App ID、App Secret、飞书表格链接、处理人姓名后即可使用。
- 后端通过 App ID / App Secret 换取 tenant_access_token。
- 读取、领取、动态同步、审核回写全部使用应用身份访问表格。

## 必须满足

飞书表格必须把该应用加入文档权限，并给应用可编辑权限，否则应用身份无法读写该表。

## 保留优化

- 基础数据流式加载。
- 动态同步 60 秒周期，有变更 30 秒后再同步。
- 动态同步只同步：处理人 / 审核状态 / 审核备注 / 审核时间。
- 审核回写约 700ms 触发，1.5 秒兜底检查。
- 单线程动态同步调度，带超时和失败退避。
- 保留原审核 UI、图片预览、处理人筛选、Excel 合并按钮、领取任务、不通过快捷原因、不通过原因不能为空校验。

## Cloudflare Pages

- Build command 留空
- Build output directory 填 public


## 本次本地/线下模式修复

- 「同步数据源」按钮保留显示，不隐藏。
- 线下文件模式下，顶部不再显示后台动态同步周期、下次倒计时等同步计时，只显示“线下模式：本地复审进度实时保存”。
- 线下文件模式点击「同步数据源」时，如果未配置飞书数据源，只提示“无需手动同步”，不会启动计时。
- 导出 Excel 改为自定义命名保存：优先使用系统保存窗口；不支持时弹出文件名输入框，再按命名下载。
- 修复本地无法直接读取真正二进制 `.xlsx` 的问题：旧版本虽然允许选择 `.xlsx`，但实际按文本读取；新版增加 XLSX 二进制解析。
- 本地读取增强：支持 `.xlsx / .xls / .csv / .tsv / .xml / .html / .json / .ndjson`，兼容 Excel 另存为 XML/HTML/CSV 或改扩展名后的文件。
- 自动选择有效工作表，自动识别表头行，跳过表头前空行。
