数据源设置

v3 Preview 已去掉默认内置数据。当前支持本地文件、飞书云文档、飞书 Sheets、飞书 Bitable 四种选项模式。 飞书支持浏览器直连：每个人填写自己的 App ID / App Secret，并直接粘贴飞书链接。正式大范围部署时再切换代理模式也可以。

本地文件

点击顶部「导入 Excel」，支持 .xlsx / .xls。第一行作为表头。

飞书云文档 Doc

用于拉取云文档文本内容。适合放审核规则、说明、提示词等；如要作为审核数据源，建议文档内容使用 JSON / TSV 格式。

飞书 Sheets

只需要在上方粘贴飞书表格链接，系统会自动解析 Spreadsheet Token 和链接中当前打开的 Sheet ID，并默认读取该工作表的全部有效区域；只有链接里没有 Sheet ID 时，才兜底读取第一个工作表。第一行作为表头；表头顺序不固定，多余字段自动忽略，缺失字段按空白处理；品牌层级/品牌等级字段会保留用于筛选、展示和导出。

飞书 Bitable 多维表格

适合多人协作审核。第一版支持读取记录；若填写「当前处理人」，系统会自动筛选处理人字段等于该姓名的记录；后续可继续补审核结果回写。
